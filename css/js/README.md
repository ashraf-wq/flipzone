# FlipZone Landing Page

A responsive landing page for FlipZone, an upside-down museum where gravity disappears. Built with HTML, CSS, and JavaScript.

## Features
- Hero section with mixed RGB gradient background and cover image.
- Pricing cards in square boxes at the top.
- Separate "Explore More" slide with animated text.
- Translation button for English/Arabic.
- Smooth scrolling and animations.

## How to Run
1. Clone or download the repo.
2. Open `index.html` in a browser.
3. Customize images and content as needed.

## Technologies
- HTML5
- CSS3 (with animations)
- JavaScript
- Bootstrap 5
